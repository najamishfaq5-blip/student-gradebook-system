🎓 Student Gradebook System

This is a Python-based project developed as part of my Data Science learning journey under Hunarmand Punjab.

🚀 Features
- Add student records
- Store marks using dictionaries
- Calculate average marks
- Assign grades automatically
- View complete gradebook

🛠️ Technologies Used
- Python
- Loops
- Dictionaries
- Functions

▶️ How to Run
1. Open the project in PyCharm or any Python IDE
2. Run the file `gradebook.py`
3. Follow the menu options
